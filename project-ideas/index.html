Project Ideas

Here’s some vague project ideas, with some basic information about which paradigms are well-suited to which ideas. I’m hopeful that this helps our brainstorming!

Local-first ideas

  • an online collaborative editor for music, art, or anything else!
  • a ahare-alike / buy-nothing app for arranging trades
  • a course shopper that lets you see where your friends are enrolled
  • a social application where you can post messages and reply to threads
  • a collaborative application for reporting sightings of scooters

Serverless ideas

  • an online service for editing images in a predictable way
  • a recognition pipeline for identifying items in images
  • a continuous-integration service for compiling or checking code
  • a compression service for removing duplicates and enforcing consistent naming in a hosted storage bucket

Traditional VM ideas

Most applications can be a good fit for traditional VMs—including all of the above! We can brainstorm more about this in class if we want, but for now I’m going to keep us on the serverless and local-first buckets.
